## 功能特色

### 📚 培训资料库
- 主播话术手册（在线预览 + 下载）
- 客服话术手册（在线预览 + 下载）
- 直播培训PPT（8章完整内容）

### 📊 直播数据分析
- 实时KPI概览卡片
- 可视化图表（趋势、主播对比、时段分析）
- 主播业绩排名
- AI智能洞察
- 详细数据表格（支持导出）

### 📅 直播排班管理
- 周视图排班日历
- 智能排班建议
- 热门时段标记

### 🎯 话术评分系统
- AI智能评分
- 话术优化建议
- 改进前后对比

### 🏆 主播成长体系
- 四级成长体系（青铜→白银→黄金→铂金）
- 任务清单与积分奖励

### 🔔 实时预警中心
- 获客成本超标预警
- 观看人数下降提醒
- 小风车点击率监控
- 竞品动态追踪

### 📖 知识库系统
- 话术案例库
- 常见问题FAQ
- 行业资讯
- 热门话题排行

## 技术栈

- HTML5 + CSS3
- JavaScript (ES6+)
- Chart.js 数据可视化
- Font Awesome 图标
- GitHub Pages 部署
